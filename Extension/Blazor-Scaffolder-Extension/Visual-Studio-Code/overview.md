# Overview of Blazor UI Scaffolder Extension for Visual Studio Code

The [Syncfusion® Blazor Scaffolder Extension for Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=SyncfusionInc.Blazor-VSCode-Extensions) simplifies the process of adding views and controller action methods for Syncfusion® Blazor **DataGrid, Tree Grid, Charts, Scheduler and Diagram** controls in Blazor applications. The scaffolding UI allows you to select the type of data source binding, Syncfusion® component features and data fields for a selected control, significantly reducing the time required to develop standard operations.
